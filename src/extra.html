Diferencia entre UI y UX

Se sabe acerca del concepto alrededor de UI y UX, pero no necesariamente cuáles son las principales diferencias entre cada uno.
UI (User Interface) se refire a todo aquello que tiene que ver con la interface con la que interactua el usuario, cosas como botones, pantallas, iconos, etc.
UX (User experience) se refiere a todo aquello que tiene que ver con la experiencia del usuario, ya sea en una plataforma o un sitio web, y como estos se sienten acerca de esta interacción.
Ambos tienen un gran rol en el proceso de creación del proyecto.
